1.Calculadora de IMC Neon:
Uma calculadora de IMC (Índice de Massa Corporal) com design neon, feita com HTML, CSS e JavaScript. Permite calcular o IMC a partir do peso e altura, exibindo o resultado e a classificação de saúde.

2.Funcionalidades:
Cálculo do IMC com base em peso (kg) e altura (m);

Validação de campos;

Interface moderna com efeitos neon.

3.Classificação do IMC:
IMC	Classificação;
< 18.5	Abaixo do peso;
18.5 – 24.9	Peso normal;
25 – 29.9	Sobrepeso;
≥ 30	Obesidade.

4.Como usar

Abra index.html no navegador;

Preencha Nome, Altura e Peso;

Clique em Calcular para ver o resultado.

4.Tecnologias

HTML5;

CSS3 (efeito neon);

JavaScript.
